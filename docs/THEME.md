# 解決したい課題 / 潜在的なニーズ

- ガンプラが好きで買いたい気持ちはあるが、何を持っているか・どれだけ積んでいるかが把握できておらず、気づいたら積みプラが増え続けている。
- 予算は限られているのに「欲しい」という気持ちに負けて後悔する買い物をしてしまいがち。
- また制作中に流せるコンテンツも毎回自分で探す手間がかかっている。

# 対象ユーザー（誰の、どんな状況か）

- 転売目的ではなく、純粋にガンプラを楽しんでいる社会人モデラー(学生も可)。
- 毎月使える予算を決めて、その範囲内で賢く・後悔なく購入したいと思っている人。全部は買えないからこそ、限られた予算の中で何を優先するか真剣に悩んでいる。

# プロダクトのカテゴリー

- ガンプラ・ホビー管理ツール(コレクション管理 × AI購入アドバイザー)

# 提供する価値（ユーザーが得られる変化・体験）

- 積みプラ・予算状況が可視化されるので「今の自分の状態」が一目でわかる
- AIと対話しながら購入を検討できるので、衝動買いではなく納得感のある買い物ができる
- 制作中に流せるコンテンツをAIが提案してくれるので、作る時間がより豊かになる

# 価値を実現するための最小限の機能（MVP）

- これ持ってたっけ？の解消: 所持・積みプラの登録・管理(写真アップロードもできると嬉しい)
- 一目でわかるお財布状況: 月の予算設定・使用額の記録・残額の表示
- オタク仲間のような感覚で相談できる: AIとの対話型購入アドバイス(「今月MGをもう一個買うべきか」「予算を削ってでもこれを買う価値があるか」など自由に相談できる)、また、購入候補のガンプラを入力すると、AIが過去の購入履歴や予算状況を踏まえてアドバイスしてくれる。キットごとのAIレビュー（いい点・悪い点）もあり。
- DJ AI MATSUGANO: 制作中に流せるYouTube動画をAIが提案(YouTube Data API活用)
- ユーザーレビューの投稿・閲覧・星評価
- ユーザープロフィールの作成・編集(ニックネーム、好きなシリーズなど)
- ガンプラの検索機能(シリーズ、スケール、発売年などで絞り込み)
- キットのマスタデータはユーザーが登録・蓄積していく（コミュニティ型）

# 既存の代替手段・競合との違い

- 入荷NAVIとの違い: 再販・新作情報の通知に特化しており、「買った後の管理」や「買うべきかの判断サポート」はできません。本アプリは買う前・買った後を一気通貫でサポートします。
- おきちゃんのガンプラ堂との違い: 再販カレンダーや情報まとめに特化したブログメディアであり、自分の所持状況や予算と紐づけた管理はできません。本アプリは自分のデータをベースにパーソナライズされたアドバイスを受けられます。
- スプレッドシート・メモ管理との違い： 自己管理ツールとして使っている人もいますが、AIとの対話や購入アドバイスとの連携はできません。本アプリは「記録する」だけでなく「次のアクションを一緒に考える」体験を提供します。

# 技術的にこだわりたい・工夫したい点（無理ない範囲で）

- Anthropic APIを活用した対話型購入アドバイス・動画提案機能
- YouTube Data APIとの連携
- ユーザーフレンドリーなUI/UX設計(特にスマホでの使いやすさ)
- ガンプラのマスタデータはユーザーが登録・蓄積していくコミュニティ型の仕組みを工夫して、ユーザー同士の交流も促進したい
- 丁寧なDB設計と読みやすいコードで、開発・保守しやすいプロダクトを目指す
- レスポンシブデザインで、PC・スマホどちらでも快適に使えるようにする

# なぜ自分がこれを作るのか（動機・原体験）

- 稼いだお金をどこに使うか、我慢ばかりもしたくないが無駄遣いもしたくない、という自分自身のテーマがありました。スキルへの投資とガンプラへの出費のバランスをどう取るか、それは自分をコントロールする力であり、エンジニアとして生きていく上でも必要な姿勢だと思っています。このアプリはそのための自分自身のツールでもあると思っています。
